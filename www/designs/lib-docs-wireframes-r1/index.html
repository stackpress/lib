API article

EventEmitter

Article pages should stay calm and highly scannable: import first, when-to-use next, API tables, example, related references.

Import

import { EventEmitter } from '@stackpress/lib';

When to use it

Use this area for a concise decision rule, not a tutorial. Experienced users should know whether they are in the right place within a few seconds.

API

MethodReturnsNotes
emit()Promise statusExecutes matching listeners in queue order.
on()thisAdds a priority-aware listener.
use()thisMerges listeners from another emitter.

Example

const emitter = new EventEmitter();

emitter.on('saved', async id => {
  console.log('saved', id);
}, 10);

await emitter.emit('saved', 'abc123');
